iCodeMash Schedule 2010

If you're like me, you may be overwhelmed by all the great sessions planned for this year's CodeMash. Last year, I found myself picking sessions to attend at the very last minute. I'd flip through the paper schedule to see what's next, what it's about, and where to find it.

Hey, it's 2010! Paper schedules are so last decade.

This web app can be run offline and installed to your iPhone/iPod Touch home screen by using the "+" button at the bottom of Mobile Safari. You might be able to get this to work on other WebKit based browsers, but I haven't tested it. The app can tell if it's been updated and will automatically download new versions of itself.

You're using an HTML5 Offline Web application customized for the iPhone/iPod Touch by the jQTouch plugin for jQuery. It's served up to you by a Ruby/Rack application on Heroku.

You can get the source code on github.

Feel free to contact me if you have any questions or if you notice anything amiss.
